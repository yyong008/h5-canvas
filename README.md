## canvas 的api以及使用方式

### 绘制矩形

storkeRect(x, y, width, height)
clearRect(x, y, width, height) // 清除的完全是透明的

### 绘制路径

beginPath()
moveTo(x, y)
closePath()
stroke()
fill()
